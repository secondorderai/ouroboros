# Desktop How-To Guides

These guides cover the user-facing features in the Ouroboros desktop app. They
focus on what to click, when to use each feature, and the constraints visible in
the app.

They assume you are using the published macOS release of `Ouroboros.app`, not
running Ouroboros from a Git repository.
