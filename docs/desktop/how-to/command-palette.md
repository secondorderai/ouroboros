# Use The Command Palette

## What This Is For

The command palette is a fast way to start common actions, navigate to drawers,
and open specific settings sections.

## When To Use It

Use the command palette when you want to avoid hunting through the title bar,
sidebar, or Settings navigation.

## Steps

1. Press `Cmd+K` on macOS or `Ctrl+K` on Windows and Linux.
2. Type to search actions.
3. Use `ArrowDown` and `ArrowUp` to change selection.
4. Press `Enter` to run the selected action.
5. Press `Escape` or click outside the palette to close it.

## Try It

Open the palette and search for:

```text
new
```

```text
dream
```

```text
appearance
```

Then start a chat with:

```text
Give me three useful things I can try in this app in the next five minutes.
```

## Available Actions

- `New conversation`
- `Trigger dream cycle`
- `Open workspace folder`
- `Browse skills`
- `View evolution log`
- `Team graph`
- `Approvals queue`
- `Change model`
- `Configure permissions`
- `Manage skills`
- `Manage API keys`
- `Appearance`
- `Modes`

## Constraints And Gotchas

- Empty searches show `No matching actions`.
- Some actions open a drawer or Settings section instead of changing the current
  chat directly.
- `Open workspace folder` selects a folder and switches the store to
  `Workspace` mode for a new empty chat.
- The palette closes before running the selected action.
