# Manage Sessions

## What This Is For

Sessions are conversation records in the left sidebar. They let you return to
previous chats, create new chats, rename useful threads, and delete old ones.

## When To Use It

Use sessions when you want separate conversations for separate tasks, or when
you need to resume earlier work without mixing context into the current chat.

## Steps

1. Click the sidebar button in the title bar to show or hide the session list.
2. Click `New conversation` in the sidebar to start a new session.
3. Click any session in the list to load it.
4. Right-click a session to open the session menu.
5. Choose `Rename`, enter the new title, and click `Save`.
6. Choose `Delete`, then confirm with `Delete` to remove a session.
7. If the list has more history, click `Load More`.

## Try It

Create separate sessions with prompts like:

```text
Help me plan meals for the next three days using simple ingredients.
```

```text
Help me draft a friendly message to reschedule a coffee meetup.
```

```text
Help me brainstorm low-cost ways to make my apartment feel more organized.
```

Then rename each session with a short title such as `Meal plan`, `Coffee
message`, or `Home reset`.

## Constraints And Gotchas

- Sessions are grouped by `Today`, `Yesterday`, `This Week`, and `Older`.
- A running session shows a processing indicator and a status such as
  `Running command`, `Reading files`, or `Working`.
- Deleting a session cannot be undone.
- The sidebar can be resized by dragging its right edge.
